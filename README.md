# 🏥 MedBridge Clinic Website

A responsive, accessible multi-page clinic website built with vanilla HTML, 
CSS, and JavaScript. Designed to help patients find services, learn about 
the clinic, and get in touch easily.


## ✨ Features

### 🏠 Home Page
- Hero section with clinic tagline and call-to-action buttons
- About section highlighting faster scheduling, patient resources, and support
- Responsive layout that works on mobile and desktop

### 🩺 Services Page
- Full list of clinic services and features
- Interactive image gallery with thumbnail switching
- FAQ section with toggle show/hide functionality

### 📬 Contact Page
- Clinic information — address, phone, email, hours
- Contact form with full client-side validation
- Validates name length, email format, and message length
- Success and error status messages on submission

### ♿ Accessibility
- Skip-to-content link for keyboard users
- ARIA labels on all interactive elements
- Semantic HTML throughout (header, nav, main, footer, article)
- Mobile-friendly hamburger navigation using CSS checkbox toggle

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Responsive Design (mobile-first)
- Accessibility best practices (WCAG)

## 🚀 Getting Started
```bash
# Clone the repository
git clone https://github.com/yanetassefa1/medbridge-clinic-website

# Open in your browser
open index.html
```
No dependencies or build tools required — just open `index.html` in any browser.

## 📁 Project Structure
```
medbridge-clinic-website/
├── index.html       # Home page
├── services.html    # Services + image gallery + FAQ
├── contact.html     # Contact form + clinic info
├── styles.css       # All styling
└── script.js        # Gallery, FAQ toggle, form validation
```
